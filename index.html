Атрибуты доступности

aria-label - создаёт текстовую метку текущего элемента в случае отсутствия видимого текста
aria-labelledby - создаёт связь между объектами и их метками
alt - текстовое описание изображения

И что делать?

Ориентиры

banner - содержит ориентированный на сайт контент каждой страницы, например логотип, обычно расположенный вверху страницы. Только один экземпляр на странице
navigation - навигационные ссылки для навигации по сайту или странице
main - основное содержимое. Только один экземпляр на странице
contentinfo - содержит контент, который обычно находится в нижнем колонтитуле страницы, например заявления об авторских правах и конфиденциальности. Только один экземпляр на странице
